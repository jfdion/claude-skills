# Projet de recherche - Analyse comparative
## Méthodologie de recherche - Automne 2025

---

## Description du projet

Vous devez réaliser une analyse comparative sur un sujet de votre choix. Le travail est individuel et vous devrez le présenter à l'équipe lors de la séance finale. La remise se fait sur la plateforme avant la date limite.

## Objectifs

L'objectif principal est de démontrer votre maîtrise des concepts vus en classe. Vous devez appliquer les méthodes de base de l'analyse comparative.

## Méthodologie

### Cadre théorique
Utilisez une approche rigoureuse et assurez-vous d'avoir une bonne qualité de recherche. Le travail doit être solide et bien structuré.

### Éléments requis

#### Revue de littérature
- Identifier les sources pertinentes avec leurs informations
- Elles doivent être organisées de manière appropriée
- Analyser les théories disponibles

#### Collecte de données
- Les participants peuvent être recrutés de plusieurs façons
- Gérer les aspects éthiques avec soin
- Documentation disponible

### Sources
Vous pouvez utiliser n'importe quelle base de données pour vos recherches. Assurez-vous que ça fonctionne bien.

## Évaluation

### Critères de notation (100 points)

**Méthodologie (40%)**
- Approche claire et organisée
- Bonne utilisation des méthodes

**Contenu (40%)**
- Tous les éléments principaux inclus
- Analyse stable

**Présentation (20%)**
- Document de qualité
- Texte bien rédigé

### Bonus
Un bonus peut être accordé pour travail exceptionnel.

## Modalités de remise

### Format
Remettez le tout dans un format approprié avec l'analyse et la documentation.

### Délai
Date limite: fin de session
Pénalité en cas de retard

### Présentation
Vous devrez faire une présentation de votre projet. Préparez quelques exemples pour la présentation.

## Ressources

Référez-vous aux notes de cours et à la documentation fournie. Si vous avez des questions, contactez-moi.

## Politique sur l'IA

L'utilisation d'IA générative est permise pour vous aider si vous êtes bloqué, mais vous ne pouvez pas l'utiliser pour générer le contenu principal.

## Travail en équipe

Bien que le travail soit individuel, vous pouvez discuter des concepts avec vos collègues. L'équipe doit collaborer lors de la présentation finale.

## Critères spécifiques

### Longueur du travail
Le document doit faire au moins 15 pages. Assurez-vous d'inclure suffisamment d'exemples.

### Citations
Suivez les normes de citation appropriées. Les références doivent être complètes.

### Annexes
Incluez les données brutes dans les annexes. Organisez-les correctement.

---

**Note:** Tous les étudiants doivent respecter les règles institutionnelles concernant le plagiat et l'intégrité académique.
